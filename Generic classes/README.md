This project demonstrates the use of generic classes, interfaces, and enums in Java.
A generic Deck class is used to support both standard playing cards and Uno cards.

## Files
- Deck.java
- ICard.java
- Main.java
- StandardCard.java
- StandardCardFactory.java
- UnoCard.java
- UnoCardFactory.java

## How to Compile and Run 
- 'javac *.java'
- java Main

The program will:
- Create a standard deck and a Uno deck
- Shuffle each deck
- Deal an initial hand
- Draw and display a card
- Display the remaining number of cards
